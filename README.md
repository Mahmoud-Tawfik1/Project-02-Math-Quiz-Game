# 🧮 Project 02 — Math Quiz Game (C++)

This repository contains my implementation of a **Math Quiz Game** using **C++**.

The project was developed as part of the **Programming Advices** learning track by **Dr. Mohammed Abu-Hadhoud** to practice structured programming and build a complete interactive console application.

---

## 🚀 About the Project

The purpose of this project was to apply the programming concepts learned throughout the course by building a quiz game that generates random math questions.

While working on this project, I practiced:

- Structured programming using C++
- Organizing code into reusable functions
- Using enums and structs
- Working with random number generation
- Managing game flow and player results

---

## 🎯 Project Features

The game allows the player to:

- Choose the number of questions
- Select the difficulty level:
  - Easy
  - Medium
  - Hard
  - Mixed
- Select the operation type:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mixed
- Solve randomly generated math questions
- Receive immediate feedback after each answer
- View a final quiz summary showing:
  - Total questions
  - Correct answers
  - Wrong answers
  - Final result (Pass or Fail)
- Restart the game after finishing

## 🕹️ How It Works

1. The player chooses:
   - Number of questions
   - Difficulty level
   - Operation type

2. The program generates random arithmetic questions.

3. The player enters an answer for each question.

4. The program checks the answer and displays the result immediately.

5. After all questions are completed, the final quiz statistics and overall result are displayed.
